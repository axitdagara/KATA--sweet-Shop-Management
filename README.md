# Sweet Shop Management

Welcome to the Sweet Shop Management project! This project focuses on managing a sweet shop's inventory, sales, and customer interactions.

## Features
- **Inventory Management**: Keep track of all the sweets available in the shop.
- **Sales Tracking**: Record sales and generate reports.
- **Customer Management**: Manage customer information and orders.
